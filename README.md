# Node akf-lang Interpreter
A very basic language. 

## Usage:
1 - `node index.js ./path/to/your/file.akf`

## Functions:
- write
- writeln
- take (Run other akf file (beta))
- var
- multi
- div
- plus
- less

## Variable types:
- bool
- num
- null
- str
